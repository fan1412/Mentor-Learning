## 如何评价PI参数—相关的指标

这个式子本质上是**电枢回路的基尔霍夫电压定律**加上**线圈电感效应**和**反电动势**得到的：

\[  V=L\frac{di}{dt}+Ri+K_e\omega  ]\



它常见于**直流电机**或把交流电机某一轴等效后得到的**电机绕组电压方程**。

* * *

1. 先看每一项分别表示什么
   ==============

[  
V=L\frac{di}{dt}+Ri+K_e\omega  
]

含义是：外部加到绕组上的电压 (V)，被三部分“消耗”掉了：

1. **电感压降**  
   [  
   L\frac{di}{dt}  
   ]  
   因为绕组本身是电感，电流变化时会产生感应电压，阻碍电流变化。

2. **电阻压降**  
   [  
   Ri  
   ]  
   绕组导线有电阻，电流流过就有欧姆压降。

3. **反电动势**  
   [  
   K_e\omega  
   ]  
   电机转起来后，导体切割磁场，绕组中会感应出一个电压，这个电压方向与外加电压“对抗”，所以叫反电动势。

* * *

2. 这个式子怎么推导出来
   =============

2.1 从绕组回路列 KVL
--------------

对电机电枢绕组或某相绕组看成一个回路，按基尔霍夫电压定律：

[  
V = v_R + v_L + e  
]

其中：

* (v_R = Ri)

* (v_L = L\frac{di}{dt})

* (e) 是电机旋转产生的感应电动势

于是：

[  
V = Ri + L\frac{di}{dt} + e  
]

所以关键就在于说明为什么

[  
e = K_e\omega  
]

* * *

2.2 为什么反电动势与转速成正比
-----------------

根据**法拉第电磁感应定律**：

[  
e = \frac{d\lambda}{dt}  
]

其中 (\lambda) 是磁链。

对于一个转动电机，绕组所链接的磁链会随着转子位置变化。设转子机械角位置为 (\theta)，那么磁链可写成 (\lambda(\theta))。于是：

[  
e = \frac{d\lambda}{dt}  
= \frac{d\lambda}{d\theta}\frac{d\theta}{dt}  
]

而

[  
\frac{d\theta}{dt}=\omega  
]

所以：

[  
e=\frac{d\lambda}{d\theta}\omega  
]

如果磁场分布、绕组结构都固定，那么 (\frac{d\lambda}{d\theta}) 可视为常数，于是：

[  
e=K_e\omega  
]

这就得到反电动势项。

* * *

3. 更直观地理解
   =========

电阻项 (Ri)
--------

很好理解，就是导线发热损耗。
电感项 (L\frac{di}{dt})
--------------------

绕组像一个“惯性元件”，不允许电流突然变化。你想让电流快速上升，就必须额外施加电压去克服这个“阻碍”。
反电动势项 (K_e\omega)
-----------------

电机越快，导体切割磁力线越快，感应电压越大。  
所以转速越高，想继续把电流压进去就越困难。

这也是为什么电机空载时，速度上去以后电流会变小：因为大部分外加电压都被反电动势抵消了。

* * *

4. 也可以从能量角度理解
   =============

电机是电能和机械能转换装置。

输入电功率：

[  
P_e = Vi  
]

其中一部分变成铜耗：

[  
P_R = Ri^2  
]

一部分暂时储存在磁场中：

[  
P_L = iL\frac{di}{dt}  
]

还有一部分转成机械功率：

[  
P_m = ei  
]

而机械功率又有：

[  
P_m = T\omega  
]

所以电机里那个“和机械运动有关的电压” (e)，自然应该和 (\omega) 有关。在线性区里就是：

[  
e=K_e\omega  
]

* * *

5. 这个式子通常适用于什么电机
   ================

它最典型地用于：

* **直流有刷电机**

* **无刷直流电机 BLDC 的等效单相模型**

* **永磁同步电机 PMSM 在 dq 坐标系下的简化形式**

* 其他电机绕组的局部等效模型

对于不同电机，形式会稍微复杂一些，但核心思想都一样：

[  
\text{端电压}=\text{电阻压降}+\text{电感压降}+\text{运动感应电压}  
]

* * *

6. 如果是直流电机，还能和力矩方程配套
   ====================

电气方程：

[  
V=L\frac{di}{dt}+Ri+K_e\omega  
]

机械方程：

[  
J\frac{d\omega}{dt}=K_t i - T_L - B\omega  
]

其中：

* (J)：转动惯量

* (K_t i)：电磁转矩

* (T_L)：负载转矩

* (B\omega)：粘性阻尼

这两式合起来就是电机最基本的动态模型。

* * *

7. 为什么有时写成 (V=Ri+L\dot i+e_b)
   =============================

这是同一个东西，只是记号不同：

* (\dot i = \frac{di}{dt})

* (e_b) 表示 back EMF（反电动势）

* 若 (e_b = K_e\omega)，就得到你写的式子

* * *

8. 一句话总结
   ========

这个方程是这样来的：

1. 把电机绕组看成一个电阻 + 电感回路；

2. 电机转动时，根据电磁感应会产生反电动势；

3. 用 KVL 列回路电压平衡：

[  
V = Ri + L\frac{di}{dt} + K_e\omega  
]

所以它不是凭空写出来的，而是**电路定律 + 电磁感应定律**的直接结果。
